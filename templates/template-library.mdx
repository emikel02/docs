# Template Library

The Template Library contains 17 professionally designed templates to get you started quickly.

## Accessing the Library

Click **Template Library** in the left sidebar to browse all system templates.

## How to Use Library Templates

<Steps>
  <Step title="Browse by Category">
    Filter templates by Cold Outreach, Lead Qualification, Value Proposition, or Personalization
  </Step>
  <Step title="Preview a Template">
    Click any template to see its prompt and configuration
  </Step>
  <Step title="Duplicate to Customize">
    Click **Duplicate** to copy it to your Templates
  </Step>
  <Step title="Customize and Save">
    Edit the duplicated template to match your needs
  </Step>
</Steps>

<Note>
  System templates are read-only. You must duplicate them to make changes.
</Note>

## Template Library vs My Templates

| Template Library | My Templates |
|------------------|--------------|
| System templates (17) | Your custom templates |
| Read-only | Full edit/delete |
| Click "Duplicate" to use | Create from scratch |
| Inspiration source | What you use in jobs |

<Tip>
  Start with a Library template and customize it. Much faster than building from scratch!
</Tip>
